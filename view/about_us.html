Thanks to Dan Menard

After reading several books, and viewing several MVC patterns online, I decided I liked the pattern as described in the book Instant AngularJS Starter by Dan Menard at Amazon . It places all of the code in the folders named model, view, and controller . I did not want to throw every java script into a js folder. In this app, I did add a fourth folder called lib, for angular.js and bootstrap.js run time scripts.

Coder for this site : Kajackdfw

This demo was written by kajackdfw . Having a background in JQuery UI design, I did not want to be dependant on plugins or libraries versions that may get ported at irregular or infrequent intervals, compared to the AngularJS project. I developed this tabbed interface using AngularJS Methods in my page routing code and plain Twitter Bootstrap. Both stock AngularJS page routing and JQuery tabbed interfaces use the # symbol in the url, so my method in this demo integrates tab controlls and page urls together, with no url conflicts or JQuery.
